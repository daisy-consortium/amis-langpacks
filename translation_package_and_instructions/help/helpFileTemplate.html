Adaptive Multimedia Information System

What is AMIS?

AMIS is a software program that you can use to read DAISY Digital Talking Books. It is self-voicing, meaning that no specialized screen-reading software is needed in order for it to be used by visually impaired people. AMIS is open source software and is provided free of charge. AMIS is created and supported by the DAISY for All Project, which is funded by the Nippon Foundation. AMIS is available in many languages. Visit our website for language pack downloads. Please continue reading to learn more about the features of AMIS.



What is DAISY?

DAISY stands for Digital Accessible Information SYstem. DAISY books are fully accessible digital talking books. Common features of books, such as sections, pages, footnotes, sidebars, and bookmarks are represented in a way that allows users to quickly navigate using them. You can learn more about DAISY at http://www.daisy.org.



Quick Start

This section is intended to give you a quick list of commands to help you use AMIS.

While Reading a Book

Getting Help: Press F1 at any time to load this Help book.

Loading a CD-ROM: Press Control + L to load the contents of a CD-ROM.

Open a file on your hard drive: Press Control + O to open a file.

Switch focus to the navigation window: Press Control + N to go to the navigation window. Then use the arrow keys to move around the lists of navigation items.

Playing and Pausing the book: Press Space.

Exiting AMIS: Press Alt + F4.

Switch focus to the main text window: Press Control + A

Increase section depth granularity: Press Control + P

Decrease section depth granularity: Press Control + Q

In Dialog Screens

Hear the instructions for a dialog: Press Control + I.

Hear the text you entered: Press Control + T. Applies only if the dialog contains editable text.

Repeat the last prompt: Press Control + R.

The Sidebar

The sidebar, pictured above, contains navigation items such as sections and lists of items such as pages, footnotes, and more. These items are organized into lists. Each list has its own tab.

A special keyboard shortcut to remember is Control + N, which gives focus to the sidebar and lets you navigate whatever the current list is. You can change lists by using the view menu.

Add-Ons

AMIS supports customization via add on packages, such as plug-ins and language packs.

Plug-ins

AMIS supports plug-ins, which are alternate software-only or software plus hardware interfaces. Joysticks, refreshable Braille displays, and scanning menus are all examples of plug-ins. All plug-ins are available for download from our website (http://amis.sf.net).

Language Packs

AMIS supports UNICODE and can be configured to use different language packs. All language packs can be downloaded from our website (http://amis.sf.net).
