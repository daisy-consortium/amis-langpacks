AMIS

Adaptive Multimedia Information System

Website: http://amis.sf.net



AMISについて

AMISはDAISY録音図書の再生用ソフトウェアです。音声読み上げ機能を持っているので、スクリーンリーダーは必要ありません。AMISはオープンソース・ソフトウェアであり、無料で提供されます。AMISは、DAISY for All Projectにより開発され、サポートされます。(DAISY for All Projectは日本財団より助成を受けています。)。AMISは多くの言語をサポートしています。 AMISのWEBサイトより言語パックをダウンロードしてください。AMISの機能については以下を参照して下さい。



DAISYについて

DAISYはアクセシブルな情報システムの規格です。 DAISY録音図書は誰もが理解しやすいデジタルトーキングブックです。 DAISYを使用することで、セクション、ページ、脚注、サイドバー、ブックマークなどですばやく操作することができます。 DAISYに関しての詳しい情報は http://www.daisy.org を参照して下さい。



クイック・マニュアル

AMISを使うための基本的なコマンドリストを紹介します。

図書を再生する

ヘルプを表示する：　「F1」

CD-ROMから図書を読み込む：　「Ctrl + L」

ハードディスクから図書を読み込む：　「Ctrl + O」

ナビゲーション画面を選択する：　「Ctrl + N」 選択後、矢印キーで項目を移動

再生と停止： 「Space」

AMISを終了する：　「Alt + F4」

メインテキスト画面を選択する：　「Ctrl + A」

セクションのレベルを下げる：　「Ctrl + P」

セクションのレベルを上げる：　「Ctrl + Q」

ダイアログ

開いているダイアログの説明：　「Ctrl + I」

入力したテキストを読み上げる：　「Ctrl + T」（入力可能なテキストボックスがある場合のみ）

音声読み上げの繰り返し：　「Ctrl + R」

追加機能

AMISは言語パックやプラグインなどの追加機能をサポートしています。

プラグイン

AMISはプラグインをサポートしています。プラグインはソフトウェアのみ、またはソフトウェアとインターフェース機器から構成されます。例えばジョイスティック、点字ディスプレイ、スキャニングメニューがあります。AMISのWEBサイト(http://amis.sf.net)よりダウンロードすることができます。

言語パック

AMISはUNICODEをサポートしているので複数の言語パックを使用することができます。AMISのWEBサイト(http://amis.sf.net)よりダウンロードすることができます。
