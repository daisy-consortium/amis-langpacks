AMIS

Adaptive Multimedia Information System

Hemsida: http://amis.sf.net



Vad är AMIS?

AMIS är en programvara som du kan använda för att läsa digitala DAISY-talböcker. Den har egen talsyntes, det vill säga det behövs ingen extra programvara för synskadade användare. AMIS är ett program med öppen källkod och tillhandahålls utan kostnad. AMIS har skapats och stötts av projektet DAISY for All, som bekostas av the Nippon Foundation. AMIS finns tillgängligt på många språk. Besök vår hemsida för att ladda ner språkpaket. Fortsätt läsa för att lära dig mer om AMIS.



Vad är DAISY?

DAISY står för Digital Accessible Information SYstem. DAISY-böcker är helt tillgängliga digitala talböcker. Saker som är gemensamma för böcker, såsom avsnitt, sidor, fotnoter, marginaler and bokmärken återges på ett sätt som gör det möjligt för brukaren att söka sig fram i dem. Du kan ta reda på mer om DAISY på http://www.daisy.org.



Snabbstart

Detta avsnitt ger dig en lista på snabbkommandon som hjälper dig att använda AMIS.

När du läser en bok

För att få hjälp: Tryck F1 när du vill för att öppna denna Hjälpfil.

För att ladda en CD-ROM: Tryck Control + L för att läsa innehållet på en CD-ROM.

Öppna en fil som finns på din hårddisk: Tryck Control + O för att öppna en fil.

Växla till navigationsfönstret: Tryck Control + N för att gå till navigationsfönstret. Använd sedan piltangenterna för att förflytta dig mellan punkterna i navigationsfönstret.

För att spela upp och för att göra paus i uppspelningen av en bok: Tryck Mellanslag.

Avsluta AMIS: Tryck Alt + F4.

Växla till textens huvudfönster: Tryck Control + A

Öka antalet synliga nivåer i marginaltext: Tryck Control + P

Minska antalet synliga nivåer i marginaltext: Tryck Control + Q

I dialogrutor

Lyssna på instruktionerna till en dialogruta: Tryck Control + I.

Lyssna på texten: Tryck Control + T. Gäller bara om dialogrutan innehåller redigerbar text.

Upprepa det senaste kommandot: Tryck Control + R.

Marginaltext

Marginaltexten, som syns ovan, innehåller navigationspunkter så som sektioner och listor över t.ex. sidor, fotnoter och annat. Dessa element är ordnade i listor. Varje lista har en egen flik.

Ett kortkommando att lägga på minnet är Control + N, som fokuserar på marginaltexten och låter dig navigera i den aktuella listan. Du kan byta lista genom att använda Visa-menyn.

Marginaltext på och av

Marginaltexten kan tas fram och tas bort. Använd menyn eller klicka på "x" i övre högra hörnet på marginaltextfönstret.

  • $$: $$ -> $$

Docka marginaltexten

Marginaltexten kan flyttas runt på skärmen genom att man klickar på och drar i dess namnlist. Drag den intill en kant för att docka den till vänster, höger, upptill eller nertill.

DAISY-böcker kan innehålla överhoppbara inslag. Ett överhoppbart inslag är något som du kan välja eller välja bort i boken. Exempel på möjliga överhoppbara inslag är sidnummer och fotnoter. Tillägg

AMIS stöder anpassning via tilläggspaket såsom plug-ins och språkpaket.

Plug-ins

AMIS stöder plug-ins: gränssnitt som är antingen mjukvarubaserade eller en kombination av mjukvara och hårdvara. Joysticks, punktskriftsdisplayer och scanning-menyer är olika exempel på plug-ins. Alla plug-ins finns att ladda ner från vår hemsida (http://amis.sf.net).

Språkpaket

AMIS stöder UNICoDE och kan konfigureras för att använda olika språkpaket. Alla språkpaket kan laddas ner från vår hemsida (http://amis.sf.net).
