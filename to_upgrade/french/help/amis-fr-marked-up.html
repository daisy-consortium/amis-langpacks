AMIS

Système d'information multimédia adaptatif
(Adaptive Multimedia Information System)

Site web : http://amis.sf.net/

À propos d'AMIS

AMIS est un logiciel pour la lecture de livres parlés numériques DAISY. AMIS dispose de son propre mode de lecture vocale (self-voicing), c'est-à-dire qu'il peut être utilisé par des personnes malvoyantes sans nécessiter de lecteur sonore d'écran. AMIS est un logiciel libre et gratuit. AMIS a été créé et est maintenu par le projet DAISY for All, qui est financé par la Nippon Foundation. AMIS est disponible dans de nombreuses langues ; consultez notre site Web pour télécharger des paquets de langue. Vous pouvez poursuivre la lecture de ce document pour en savoir plus sur les fonctionnalités d'AMIS.



À propos de DAISY

DAISY est l'abréviation de Digital Accessible Information SYstem (Système d'information numérique accessible). Les livres DAISY sont des livres parlés numériques totalement accessibles. Ces livres offrent des fonctionnalités de navigation rapide grâce à leur représentation des sections, pages, notes de bas de page, encadrés et marque-pages. Pour en savoir plus sur DAISY, veuillez consulter le site Web http://www.daisy.org/.



Démarrer avec AMIS

Cette section a pour objectif de vous donner un bref aperçu des commandes d'AMIS.

Lire un livre

Obtenir de l'aide : appuyez sur F1 à tout moment pour charger ce livre d'aide.

Charger un CD ROM : appuyez sur Contrôle + L pour charger le contenu d'un CD ROM.

Ouvrir un fichier sur votre disque dur : appuyez sur Contrôle + O pour ouvrir un fichier.

Passer à la fenêtre de navigation : appuyez sur Contrôle + N pour vous rendre à la fenêtre de navigation ; les touches flèche permettent ensuite de se déplacer dans la liste des éléments de navigation.

Pause : appuyez sur Contrôle + Espace pour interrompre momentanément la lecture.

Reprendre la lecture : appuyez sur Espace.

Quitter le programme : appuyez sur Alt + F4.

Écrans de dialogue

Écouter les instructions du dialogue : appuyez sur Contrôle + I.

Écouter le texte que vous avez entré : appuyez sur Contrôle + T (uniquement si le dialogue contient du texte modifiable).

Répéter la dernière demande : appuyez sur Contrôle + R.



Liste complète des fonctionnalités d'AMIS

Ouvrir un livre DAISY

Formats reconnus : DAISY 2.02 et DAISY 3.0.

Ouvrir un fichier

Charge un fichier depuis votre disque dur. Cette commande affiche un navigateur de fichiers standard à partir duquel vous pouvez sélectionner un livre DAISY.

  • Icône :
  • Raccourci clavier : Contrôle + O
  • Menu : Fichier -> Ouvrir

Charger un CD ROM

  • Icône :
  • Raccourci clavier : Contrôle + L
  • Menu : Fichier -> Charger un CD

Chercher des livres sur votre disque dur

Un dialogue dans lequel vous pouvez taper un chemin à partir duquel chercher des livres s'affiche. Les résultats de la recherche s'affichent ensuite dans la fenêtre de dialogue.

  • Icône :
  • Raccourci clavier : Alt + F pour accéder au menu Fichier, puis S
  • Menu : Fichier -> Chercher...

Faire une sélection dans votre liste de livres lus récemment

  • Icône : aucune
  • Raccourci clavier : Alt + F pour accéder au menu Fichier, puis R
  • Menu : Fichier -> Livres récents ; sélectionnez ensuite un fichier dans le menu contextuel

Préférences

Vous pouvez ajuster ici les paramètres d'AMIS, y compris le comportement au démarrage, où stocker les marque-pages, quelles couleurs utiliser, et quel paquet de langue utiliser (si vous en avez installés).

Préférences générales

  • Icône :
  • Raccourci clavier : Alt + F pour accéder au menu Fichier, puis P. L'onglet des préférences générales est sélectionné par défaut.
  • Menu : Fichier -> Préférences

Choix des couleurs

  • Icône :
  • Raccourci clavier : Alt + F pour accéder au menu Fichier, puis P. Sélectionnez ensuite l'onglet « Choix des couleurs »
  • Menu : Fichier -> Préférences, puis l'onglet « Choix des couleurs »

Choix de la langue

  • Icône :
  • Raccourci clavier : Alt + F pour accéder au menu Fichier, puis P. Sélectionnez ensuite l'onglet « Langue »
  • Menu : Fichier -> Préférences, puis l'onglet « Langue »

Vues différentes

Vue par défaut

La vue par défaut contient un menu, une barre d'outils et une barre latérale. Cette vue est similaire à de nombreuses autres applications.

  • Icône : aucune
  • Raccourci clavier : cette option est sélectionnée par défaut. Pour passer en mode de vue simple, appuyez sur F11
  • Menu : aucun

Vue simple

Cette vue n'a ni menu ni barre latérale. La barre d'outils a six grands boutons en bas de l'écran.

  • Icône : aucune
  • Raccourci clavier : F11
  • Menu : Vue -> Mode de vue simple

La barre latérale

La barre latérale (vue ci-dessus) contient les éléments de navigation comme les sections, ainsi que les listes d'éléments tels que les pages, les notes de bas de page, etc. Chaque liste a son propre onglet.

Un raccourci clavier à mémoriser en particulier est Contrôle + N, qui permet de passer à la barre latérale et de naviguer dans la liste actuelle. Il est possible de changer de liste en utilisant le menu « Vue ».

Afficher ou masquer la barre latérale

La barre latérale peut être affichée ou masquée, soit en utilisant le menu, soit en cliquant sur la croix dans le coin supérieur droit de la fenêtre.

  • Icône : aucune
  • Raccourci clavier : Alt + V pour le menu « Vue », puis S
  • Menu : Vue -> Barre latérale

Barre latérale ancrée ou flottante

La barre latérale peut être être déplacée sur l'écran en cliquant puis tirant sur sa barre de titre. Elle peut être ancrée en haut, en bas, à gauche ou à droite de la fenêtre en la plaçant près d'une bordure.

  • Icône : aucune
  • Raccourci clavier : aucun
  • Menu : aucun

Sections

  • Icône :
  • Raccourci clavier : Alt + V pour le menu « Vue » ; les touches flèche permettent ensuite de se déplacer dans les éléments section
  • Menu : Vue -> Sections

Pages

Note : cet élément peut être absent si le livre n'a pas de pages.

  • Icône :
  • Raccourci clavier : Alt + V pour le menu « Vue » ; les touches flèche permettent ensuite de se déplacer dans les éléments page
  • Menu : Vue -> Pages

Éléments spéciaux

Les éléments spéciaux sont les notes de bas de page, encadrés, notes du producteur, ainsi que les autres éléments groupés. Ils n'apparaissent dans la barre latérale d'AMIS et dans le menu Vue que s'ils sont présents dans la publication.

  • Icône :
  • Raccourci clavier : Alt + V pour le menu « Vue » ; les touches flèche permettent ensuite de se déplacer jusqu'à l'élément, dont le nom apparaît dans le menu (par exemple « notes de bas de page »)
  • Menu : Vue ; puis le nom de l'élément

Effets

Contraste des couleurs

Appuyez sur le bouton ou utilisez le raccourci clavier pour utiliser ou non le contraste des couleurs. Les couleurs utilisées peuvent être choisies dans l'onglet « Choix des couleurs » du dialogue de préférences.

  • Icône :
  • Raccourci clavier : Contrôle + C
  • Menu : Vue -> Contraste élevé

Taille du texte

Appuyez sur le bouton ou utilisez le raccourci clavier pour augmenter progressivement et dans une certaine mesure la taille du texte. Pour revenir à la taille initiale, utilisez « Vue -> Réinitialiser la taille du texte ».

  • Icône :
  • Raccourci clavier : Alt + V pour le menu « Vue », puis I
  • Menu : Vue -> Augmenter la taille du texte

Lecture

Lire

  • Icône :
  • Raccourci clavier : Espace
  • Menu : Lecture -> Lire

Pause

  • Icône :
  • Raccourci clavier : Contrôle + Espace
  • Menu : Lecture -> Pause

Volume

  • Icône : aucune
  • Raccourci clavier : Contrôle + Bas pour baisser le volume, Contrôle + Haut pour augmenter le volume.
  • Menu : Lecture -> Augmenter le volume et Lecture -> Baisser le volume

Accélérer

La vitesse de lecture augmente mais la hauteur reste inchangée.

  • Icône :
  • Raccourci clavier : Contrôle + + (Contrôle et le symbole d'addition)
  • Menu : Lecture -> Accélérer

Ralentir

La vitesse de lecture diminue mais la hauteur reste inchangée.

  • Icône :
  • Raccourci clavier : Contrôle + - (Contrôle et le symbole de soustraction)
  • Menu : Lecture -> Ralentir

Vitesse normale

Retourne à la vitesse de lecture normale.

  • Icône :
  • Raccourci clavier : Contrôle + 0
  • Menu : Lecture -> Vitesse normale

Navigation

Section suivante

  • Icône :
  • Raccourci clavier : Bas (touche flèche)
  • Menu : Navigation -> Section suivante

Section précédente

  • Icône :
  • Raccourci clavier : Haut (touche flèche)
  • Menu : Navigation -> Section précédente

Montrer la profondeur des sections

Cette commande vous permet de restreindre ou de développer complètement la hiérarchie des sections. Par défaut, quand vous ouvrez un livre pour la première fois, toutes ses sections apparaissent dans la barre latérale.

  • Icône : aucune
  • Raccourci clavier : Alt + N pour le menu « Navigation », puis S pour le menu contextuel « Afficher la profondeur des sections »
  • Menu : Navigation -> Afficher la profondeur des sections, puis choisissez une profondeur dans le menu contextuel.

Phrase suivante

Selon le style de production de la publication, une « phrase » peut en réalité être un paragraphe.

  • Icône :
  • Raccourci clavier : Droite (touche flèche)
  • Menu : Navigation -> Phrase suivante

Phrase précédente

Selon le style de production de la publication, une « phrase » peut en réalité être un paragraphe.

  • Icône :
  • Raccourci clavier : Gauche (touche flèche)
  • Menu : Navigation -> Phrase précédente

Page suivante

Les commandes de page ne sont disponibles que si la publication contient des pages.

  • Icône :
  • Raccourci clavier : Contrôle + Droite
  • Menu : Naviguer -> Page suivante

Page précédente

Les commandes de page ne sont disponibles que si la publication contient des pages.

  • Icône :
  • Raccourci clavier : Contrôle + Gauche
  • Menu : Navigation -> Page précédente

Se rendre à une page

Les commandes de page ne sont disponibles que si la publication contient des pages.

  • Icône : aucune
  • Raccourci clavier : Contrôle + G
  • Menu : Navigation -> Aller à la page...

Chercher dans le texte

Cette commande n'est disponible que si la publication contient du texte.

  • Icône :
  • Raccourci clavier : Alt + N pour le menu « Navigation », puis F
  • Menu : Navigation -> Chercher dans le texte

Sortir d'un élément échappable

Il est possible de sortir de certains éléments dits échappables tels que les structures encapsulées comme les tables, ou les structures d'information complémentaire comme les encadrés. Vous pouvez ainsi reprendre la lecture de l'élément précédent la structure échappable.

  • Icône : aucune
  • Raccourci clavier : Esc
  • Menu : Lecture -> Sortir de l'élément

Éléments facultatifs

Les livres DAISY peuvent contenir des éléments facultatifs, qu'il est possible d'ignorer pendant la lecture comme par exemple les numéros de page ou les notes de bas de page. La lecture de ces éléments peut être activée ou non ; si vous ne voulez pas qu'ils soient lus, veuillez vérifier qu'ils sont bien cochés dans le dialogue « Éléments facultatifs ».

  • Icône : aucune
  • Raccourci clavier : Alt + I pour le menu « Informations », puis S
  • Menu : Informations -> Éléments facultatifs

Marque-pages

Ajouter un marque-page

  • Icône :
  • Raccourci clavier : Contrôle + D
  • Menu : Marque-pages -> Ajouter un marque-page

Charger un marque-page

Lorsque vous ouvrez un livre, tous ses marque-pages sont automatiquement chargés. Tout nouveau marque-page sera ajouté à ce menu.

  • Icône : aucune
  • Raccourci clavier : Alt + B pour le menu « Marque-pages », puis les touches flèches pour atteindre le marque-page désiré
  • Menu : Marque-pages, puis touches flèches

Information récapitulative

AMIS propose trois types différents d'information récapitulative : sommaire complet, position courante et options de navigation, tous accessibles via un dialogue appelé par le menu « Informations » (Alt + I)

Quitter AMIS

Appuyez sur Alt + F4 pour quitter AMIS.

Ajouts

Les ajouts mentionnés ici sont disponibles à l'URL http://amis.sf.net/.

Plugins

AMIS dispose d'un mécanisme de plugins, qui peuvent être purement logiciels ou logiciels avec une interface matérielle, comme par exemple des joysticks ou des affichages régénérés en braille. Une fois un plugin téléchargé, utilisez le logiciel d'installation fourni avec le plugin, ou dézippez les fichiers dans le répertoire ..\AMIS\plugins\.

Paquets de langue

AMIS utilise Unicode et peut être configuré pour fonctionner en différentes langues. Les paquets de langue s'installent dans le répertoire ..\AMIS\lang\.
