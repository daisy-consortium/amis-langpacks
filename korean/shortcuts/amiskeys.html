키보드 숏컷

아미스와 관련있는 키보드 숏컷의 명령을 아래에 제시하였습니다. 일부 명령은 오로지 메뉴 내에서만 사용하는 낱글자 바로 가기 키입니다. 나머지 명령은 아미스의 아무 곳에서나 사용하는 다중의 바로 가기 키입니다.

메뉴로 구성

파일 (Alt+F)

  • 열기...: Ctrl+O
  • URL 열기...: Ctrl+U
  • 닫기: C
  • CD 열기: Ctrl+L
  • 드라이브 검색...: S
  • 환경 설정...: P
  • 최근: R
  • 종료: Alt+F4

보기 (Alt+V)

  • 사이드바: S
  • 기본 보기: F11
  • 페이지 스타일: P
  • 글꼴 더 크게: G
  • 글꼴 더 작게: M
  • 텍스트 스타일...: T

재생 (Alt+P)

  • 종료: Esc
  • 더 빠르게: Ctrl++
  • 더 느리게: Ctrl+-
  • 보통 속도: Ctrl+0
  • 볼륨 증가: Ctrl+Up
  • 볼륨 감소: Ctrl+Down

탐색 (Alt+N)

  • 이전 섹션: Up
  • 다음 섹션: Down
  • 섹션 단계 표시: S
  • 이전 프레이즈: Left
  • 다음 프레이즈: Right
  • 이전 페이지: Ctrl+Left
  • 다음 페이지: Ctrl+Right
  • 이동할 페이지...: Ctrl+G
  • 텍스트 찾기...: Ctrl+F

책갈피 (Alt+B)

  • 책갈피 추가: Ctrl+D

정보 (Alt+I)

  • 독서 옵션...: R
  • 간행물 적요...: P

도움말 (Alt+H)

  • 목차: F1
  • 키보드 숏컷: K
  • 아미스 정보: A

기타 (메뉴에 속하지 않음)

  • 다음 찾기: F3
  • 이전 찾기: Shift+F3
  • 사이드바에 포커스: Ctrl+N
  • 텍스트에 포커스: Ctrl+T
  • 도서 오디오 토글: Ctrl+Shift+F
  • 자체 음성 출력 토글: Ctrl+Shift+G
  • 자체 음성 볼륨 감소: Ctrl+Shift+Down
  • 자체 음성 볼륨 증가: Ctrl+Shift+Up
  • 도서 볼륨 감소: Ctrl+Shift+Alt+Down
  • 도서 볼륨 증가: Ctrl+Shift+Alt+Up
  • TTS 볼륨 증가: Ctrl+Shift+Right
  • TTS 볼륨 감소: Ctrl+Shift+Left
  • 섹션 단계 감소: Ctrl+Q
  • 섹션 단계 증가: Ctrl+P
  • 하이라이트 색상 재설정: Ctrl+Shift+E
