Συντομεύσεις πληκτρολογίου

Οι εντολές για το AMIS και οι αντίστοιχες συντομεύσεις πληκτρολογίου δίνονται παρακάτω. Μερικές εντολές έχουν συντομεύσεις μόνο ενός χαρακτήρα που πρέπει να χρησιμοποιηθούν μία φορά μέσα στο μενού. Άλλες εντολές έχουν συντομεύσεις πολλαπλών χαρακτήρων για να χρησιμοποιηθούν οπουδήποτε στο AMIS.

Οργάνωση με βάση το μενού

Αρχείο (Alt+F)

  • Άνοιγμα...: Ctrl + O
  • Άνοιγμα URL...: Ctrl + U
  • Κλείσιμο: C
  • Φόρτωση CD: Ctrl + L
  • Αναζήτηση Μονάδων Αποθήκευσης...: S
  • Προτιμήσεις...: P
  • Πρόσφατο: R
  • Έξοδος: Alt+F4

Προβολή (Alt+V)

  • Πλαϊνή μπάρα: S
  • Βασική Προβολή: F11
  • Στυλ Σελίδας: P
  • Μεγαλύτερη Γραμματοσειρά: G
  • Μικρότερη Γραμματοσειρά: M
  • Στυλ Κειμένου...: T

Αναπαραγωγή (Alt+P)

  • Escape: Esc
  • Πιο γρήγορα: Ctrl + +
  • Πιο αργά: Ctrl + -
  • Κανονική Ταχύτητα: Ctrl + 0
  • Αύξηση Έντασης Ήχου: Ctrl + Άνω βέλος
  • Μείωση Έντασης Ήχου: Ctrl + Κάτω βέλος

Πλοήγηση (Alt+N)

  • Προηγούμενη Ενότητα: Άνω βέλος
  • Επόμενη Ενότητα: Κάτω βέλος
  • Εμφάνιση Βάθους Ενότητας: S
  • Προηγούμενη Φράση: Αριστερό βέλος
  • Επόμενη Φράση: Δεξί βέλος
  • Προηγούμενη Σελίδα: Ctrl + Αριστερό βέλος
  • Επόμενη Σελίδα: Ctrl + Δεξί βέλος
  • Μετάβαση στη Σελίδα...: Ctrl + G
  • Αναζήτηση στο Κείμενο...: Ctrl + F

Σελιδοδείκτες (Alt+B)

  • Προσθήκη Σελιδοδείκτη: Ctrl + D

Πληροφορίες (Alt+I)

  • Επιλογές Ανάγνωσης...: R
  • Σύνοψη Δημοσίευσης...: P

Βοήθεια (Alt+H)

  • Περιεχόμενα: F1
  • Συντομεύσεις πληκτρολογίου: K
  • Σχετικά με το AMIS: A

Άλλο (όχι στο μενού)

  • Εύρεση επόμενου: F3
  • Εύρεση προηγούμενου: Shift + F3
  • Εστίαση στην πλαϊνή μπάρα: Ctrl + N
  • Εστίαση στο κείμενο: Ctrl + T
  • Εναλλαγή ήχου βιβλίου: Ctrl + Shift + F
  • Εναλλαγή ήχου αυτόματης ομιλίας: Ctrl + Shift + G
  • Μείωση έντασης του ήχου της αυτόματης ομιλίας: Ctrl + Shift + Κάτω βέλος
  • Αύξηση της έντασης του ήχου της αυτόματης ομιλίας: Ctrl + Shift + Άνω βέλος
  • Μείωση της έντασης ήχου του βιβλίου: Ctrl + Shift + Alt + Κάτω βέλος
  • Αύξηση της έντασης ήχου του βιβλίου: Ctrl + Shift + Alt + Άνω βέλος
  • Αύξηση έντασης του ήχου στη μετατροπή κειμένου σε ομιλία: Ctrl + Shift + Δεξί βέλος
  • Μείωση της έντασης του ήχου στη μετατροπή κειμένου σε ομιλία: Ctrl + Shift + Αριστερό βέλος
  • Μείωση βάθους ενότητας: Ctrl + Q
  • Αύξηση βάθους ενότητας: Ctrl + P
  • Επαναφορά χρωμάτων επισήμανσης: Ctrl + Shift + E
