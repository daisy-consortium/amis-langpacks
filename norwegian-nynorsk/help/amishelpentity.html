AMIS, norsk versjon

Adaptive Multimedia Information System

Nettside: http://amis.sf.net


Kva er AMIS?

AMIS er eit program som du kan bruke for å lese DAISY digitale lydbøker. AMIS er ei programvare med open kildekode, til fri bruk. AMIS er utvikla og blir støtta av "DAISY for All Project" som er finansiert av Nippon Foundation. AMIS er tilgjengeleg på fleire språk. Gå innom nettsida vår for nedlasting av språkpakker. Les vidare her for å lære meir om mogelegheitene i AMIS



Kva er DAISY?

DAISY står for Digital Accessible Information System. DAISY-bøker er digitale lydbøker, tilgjengeleg for alle. Vanlege bokfunksjonar, slik som overskrifter, sidenummerering, fotnotar, navigasjonsmenyar og bokmerke er representert slik at brukar raskt kan navigere ved hjelp av dei. Du kan lære meir om DAISY på http://www.daisy.org



Rask start

Føremålet med denne delen er å gi deg ei rask oversikt over kommandoar som hjelper deg å bruke AMIS

Medan du les boka

Få hjelp: Trykk F1 for når som helst å opne denne hjelpeboka.

Opne ein CD-ROM. Trykk Control + L for å opne innhaldet på ei CD-ROM-plate.

Opne ei fil frå harddisken din: Trykk Control + O for å opne ei fil.

For å gå til navigasjonsvindauget: Trykk Control + N. Bruk deretter piltastane for å navigere rundt i lista over navigasjonselement.

Spele av og stoppe boka: Trykk mellomromtasten og Ctrl + mellomromtasten.

Avslutte AMIS: Trykk Alt + F4.

Gå til hovudvindauget: Trykk Control + A

Auke oppløysinga i ein del av biletet: Trykk Control + P

Redusere oppløysinga i ein del av biletet: Trykk Control + Q

I dialogvindauge

Høyre instruksjonane for ein dialog: Trykk Control + I.

Høyre teksten du har skrive inn: Trykk Control + T. Fungerer berre dersom dialog inneheld redigerbar tekst.

Repeter siste handling: Trykk Control + R.

Forskjellige typar skjermbilete

Standard

Dette vindauget har meny, verktøylinjer og navigasjonsmeny. Dette liknar mange andre dataprogram og er standardvindauge i AMIS. Du kan veksle mellom dette og Basis-vindauget med hurtigtastar.

  • Tastatur Snarveg: F11

Basis-vindauget

Dette har ingen navigasjonsmeny. Verktøylinja har seks store knappar nedst på skjermen.

  • Tastatur Snarveg: F11
  • Meny: Vis -> Enkel Vising

Navigasjonsmenyen

Navigasjonsmenyen inneheld navigasjonselement som mappetitlar og liste over element som side, fotnotar mv. Desse elementa er organisert i lister. Kvar liste har si eiga arkfane.

Control + N er ein nyttig hurtigtast som flyttar markøren til navigasjonsmenyen og let deg navigere uavhengig av kvar du er i augneblinken. Du kan forandre lista ved å bruke vindaugsmenyen.

Skru navigasjonsmenyen av og på

Navigasjonsmenyen kan skruast av og på. Bruk anten meny eller trykk "x" i det øvre høgre hjørnet av menyvindauget.

  • Tastatur Snarveg: Alt+V følgt av S
  • Meny: Vis -> Rammetekst

Dokke eller flyte

Navigasjonsmenyen kan ein flytte rundt i skjermvindauget ved å klikke og dra tittelfeltet. Dra det til ein kant for å dokke det til venstre, høgre, topp eller botn.

Selektiv lesing

Eit kjenneteikn med DAISY-bøker er at dei kan innehalde mogelegheit for selektiv lesing. Eit selektivt element er eit element du kan aktivere eller deaktivere i boka. Eksempel på mogelege selektive element er sidenummer eller fotnotar. Om du ikkje ønskjer å høyre desse elementa medan du les, sørg for at dei er merka av i "Selektive element"

  • Tastatur Snarveg: Alt+I følgt av S
  • Meny: Info -> Selektive Element...

Påbyggingspakker

AMIS støttar tilpassingar via påbyggingspakker, slik som "plug-ins" og språkpakker.

Plug-ins

AMIS støttar plug-ins, som er anten berre programvare eller programvare med maskinvare grensesnitt. Navigeringsverktøy, rensbare Brailleskjermar og skannarmenyar er alle saman eksempel på "plug-ins". Alle "plug-ins" er tilgjengelege for nedlasting frå heimesida vår (http://amis.sf.net).

Språkpakker

AMIS støttar UNICODE og kan bli konfigurert til å bruke forskjellige språkpakker. Alle språkpakkene kan lastast ned frå nettsida vår (http://amis.sf.net).
