Guia de utilizador do AMIS

Este documento dá-lhe uma breve introdução ao AMIS, incluindo as suas funcionalidades, requisitos e opções. Para obter uma lista completa de todos os comandos e teclas de atalho leia o documento Teclas de Atalho no menu Ajuda.

O que é o AMIS?

AMIS significa Sistema Adaptado de Informação Multimédia.

O AMIS é um programa que pode usar para ler livros DAISY. Tem voz própria, Isto quer dizer que qualquer pessoa com deficiência visual o pode usar sem ter que utilizar um programa especializado de leitura de ecrã. O AMIS é um programa de código aberto e é disponibilizado gratuitamente. está actualmente traduzido em dezenas de linguas vá a http://amisproject.org para fazer o download da lingua que pretende.

O que é o DAISY?

DAISY significa Sistema Digital de Informação Acessível. Os livros DAISY são livros de áudio totalmente acessíveis. As funcionalidades comuns de um livro, como as secções, páginas, notas de rodapé, barras laterais e marcadores de páginas estão dispostas de forma a permitirem que o utilizador tenha acesso a uma navegação rápida.

Este documento que está a ler é um livro DAISY! Pode ver e ouvir o texto sincronizado com o áudio, e movimentar-se nele através das secções ou frases.

O DAISY tem um padrão técnico reconhecido a nível global, foi inicialmente desenvolvido para ajudar pessoas cuja deficiência visual não lhes permitia a leitura impressa, e também tem revelado um grande potencial em diversos níveis tais como tornar o texto acessível de forma geral.

O consórcio DAISY foi fundade em 1996 e consiste num número crescente de membros e organizações em todo o mundo, unidos no propósito de desenvolver meios de tornar a informação acessível a todos que tenham dificuldade em leitura impressa.

Pode saber mais sobre o DAISY em http://daisy.org.

Formatos Suportados

Os seguintes formatos de documentos são suportados pelo AMIS:

  • DAISY 2.02
  • DAISY 3 (também referido como ANSI/NISO z39.86-2005)

Requisitos de Sistema

Necessita de ter as seguintes requisitos no seu PC:

  • Windows XP, Windows Vista, ou Windows 2000
  • DirectX 9
  • SAPI 5.1 motor de texot-para-voz
  • Internet Explorer 7 (funciona com outras versões, mas recomendamos esta.)
  • 50 MB de espaço em disco

O instalador do AMIS avisará se faltar algum componente.

Esquema de Ecrã

O ecrã tem dois modo de vista: padrão e básico. O AMIS iniciará no modo de vista padrão a menos que tenha indicado para iniciar em modo básico (esta opção está na caixa de diálogo Preferências).

A vista padrão tem quatro partes principais: o menu, a barra de ferramentas, a janela de navegação á esquerda, e a janela de conteúdos no centro do ecrã.

Vista padrão do AMIS

A barra de ferramentas tem onze botões de tamanho médio: abrir um livro, carregar um cd, ir para a frase anterior, reproduzir/pausa, ir para a próxima frase, abrandar, aumentar velocidade, reduzir fonte, aumentar fonte, mudar o estilo da página, e adicionar marca na página.

A barra de ferramentas

A janela de navegação contém uma ou mais listas divididas, isto depende do conteúdo do livro. A primeira lista é sempre uma lista hierárquica das secções. A segunda lista contém os números de páginas (se o livro tiver páginas). As listas seguintes contêm items especiais como as notas de rodapé ou imagens (isto se o livro tiver algum destes items),com cada tipo de item a aparecer na sua própria lista.

A janela de navegação

A vista básica tem duas partes principais: a barra de ferramentas que está disposta no fundo do ecrã e uma janela de conteúdos no centro do ecrã. A barra de ferramentas tem quatro botões grandes: ir para a frase anterior, reproduzir/pausa, ir para a próxima frase voltar ao modo de vista padrão. No modo de vista básico não existe menu nem janela de navegação.

Funcionalidades

As secções seguintes contém breves descrições sobre as funcionalidades do AMIS organizadas por tópico.

Abrir um livro

Os livros DAISY podem estar no seu disco local, num CD ou mesmo num local remoto.

  • Abrir um livro recente
  • Abrir automaticamente no arranque o último livro lido
  • Abrir um livro a partir da linha de comandos
  • Abrir um CD com um livro ou múltiplos livros
  • Abrir um livro a partir do disco local
  • Abrir um livro a partir de uma URL

Controlar reprodução

Os livros DAISY são reproduzidos automaticamente do inicio para o fim. Eis uma lista das funcionalidades de reprdução que pode controlar:

  • Reproduzir ou fazer pausa no livro
  • Aumentar velocidade de reprodução
  • Activar opções de reprodução, como número de página, anotações e notas de radapé

Navegação

Em alternativa á navegação de reprodução, Os livros DAISY oferecem outras alternativas de navegação.

  • Anterior/Próxima frase
  • Anterior/Próxima página
  • Anterior/Próxima secção
  • Ir para a página
  • Procurar por texto

Utilizar a janela de navegação

Utilize os items da janela de navegação para pesquisar a estrutura do livro.

  • Vista hierárquica das secções
  • Lista de páginas
  • Listas para cada tipo de item especial no livro

Opções de marcador de página

  • Adicionar marcas na página (o nome é atribuido automaticamente e é baseado no que se está a marcar)
  • Ir para a marca na página
  • A lista de páginas marcadas de cada livro é carregada quando o livro for aberto.

Obter informação sobre a publicação

É fácil de obter informação sobre o livro que está a ler, basta usar a caixa de diálogo Sumário da Publicação do AMIS. Eis uma lista completa do que tem disponível:

  • Titulo
  • Autor
  • Publicado por
  • Descrição
  • Narrador
  • Duração total
  • Total de páginas
  • Data
  • Formato
  • Funcionalidades (tais como secções, páginas, e notas de radapé)
  • Página actual
  • Secção actual (incluindo o número de páginas e sub-secções)
  • Número total de items na tabela de conteúdos
  • Tamanho máximo da secção

Alterar o aspecto

O interface do AMIS permite-lhe escolher várias opções de configuração de ecrã bem como o estilo para o livro DAISY.

  • Usar modo de vista básico ou padrão
  • Mostrar/esconder a janela de navegação
  • Movimentar a janela de navegação pelo ecrã
  • Alterar o estilo da página
  • Aumentar o tamanho da fonte
  • Ajustar as cores de selecção e alterar o tipo de fonte do programa na caixa de diálogo Estilo de Texto

Defenir preferências

As seguintes definições podem ser acedidas na caixa de diálogo Preferências no menu Ficheiro.

  • Ligar ou desligar auto-voz
  • Escolher uma voz TTS
  • Ligar ou desligar selecção de texto
  • Carregar no arranque o livro mais recente
  • Iniciar o AMIS no modo de vista básico
  • Fazer pausa no AMIS automaticamente quando trocar de aplicações
  • Desactivar a protecção de ecrã enquanto usar o AMIS
  • Seleccione uma lingua para usar no AMIS

A caixa de diálogo preferências

Definir estilo de texto

As seguintes definições podem ser acedidas na caixa de diálogo Estilo de Texto no menu Ver.

  • Seleccionar uma fonte para usar no AMIS
  • Seleccionar cores de texto e de fundo para a selecção de texto durante a reproduçãoe
