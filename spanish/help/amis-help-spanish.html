AMIS

Sistema de Información Multimedia Accesible

Página web: http://amis.sf.net



¿Qué es AMIS?

AMIS es un programa para leer Libros Hablados Digitales en formato DAISY. Dispone de su propia ayuda sonora, lo que significa que no se necesita un lector de pantalla especializado para su uso por personas deficientes visuales. AMIS es un software de código abierto y se distribuye gratuitamente. Ha sido creado y mantenido por el proyecto DAISY for All, financiado por la Nippon Foundation. AMIS está disponible en muchos idiomas. Visite nuestra web para obtener el paquete de idiomas. Siga leyendo, por favor, para aprender más acerca de AMIS.



¿Qué es DAISY?

DAISY es el acrónimo de Digital Accessible Information SYstem (Sistema de Información Accesible Digital). Los libros DAISY son libros hablados digitales totalmente accesibles. Las características habituales de los libros, como secciones, páginas, notas a pie, recuadros y marcas, se representan de manera que el usuario puede recorrerlas rápidamente. Si desea mas información acerca de DAISY, visite la página del consorcio en http://www.daisy.org.



Inicio rápido

Esta parte es un compendio sencillo de los comandos de AMIS para ayudarle a utilizar el programa.

Mientras lee un libro

Obtener ayuda: Pulse la tecla F1 en cualquier momento para ver este libro de Ayuda.

Leer un CD-ROM: Pulse las teclas Control + L para cargar el contenido de un CD-ROM.

Abrir un archivo de su disco duro: Pulse Control + O para abrir un archivo.

Activar la ventana de navegación: Pulse Control + N para acceder a la ventana de navegación. Utilice luego las teclas de flechas para moverse por las listas de entradas navegables.

Reproducir y pausar el libro: Pulse la Barra Espaciadora.

Salir de AMIS: Pulse Alt + F4.

Activar la ventana de texto principal: Pulse Control + A

Ampliar el número de niveles en la ventana de navegación: Pulse Control + P

Reducir el número de niveles en la ventana de navegación: Pulse Control + Q

En pantallas de diálogo

Escuchar las instrucciones de un diálogo: Pulse Control + I.

Escuchar el texto que ha introducido: Pulse Control + T. Únicamente es aplicable si el diálogo contiene texto editable.

Repetir el último mensaje: Pulse Control + R.

Tipos de vistas

Vista por defecto

Esta vista tiene un menú, una barra de herramientas y un recuadro. Su apariencia es muy similar a la de muchas otras aplicaciones. Es la vista por defecto establecida para el AMIS. Puede pasar de ella a la vista básica con la siguiente tecla de acceso:

  • Tecla rápida: F11

Vista básica

Esta vista no dispone de menú ni de recuadro. La barra de herramientas tiene seis botones grandes colocados en la parte inferior de la pantalla.

  • Tecla rápida: F11
  • Menú: Vista -> Vista Sencilla

El recuadro

El recuadro, como se ve en la imagen anterior, contiene puntos de navegación, como secciones y listas de entradas, tales como páginas, notas a pie y otros. Estas entradas están organizadas en listas. Cada lista tiene su propia pestaña.

Una combinación especial de teclas de acceso rápido a recordar es Control + N, que activa el recuadro y nos permite navegar por la lista activa. Puede cambiar de lista utilizando el menú ver.

Ver y ocultar

El recuadro puede verse u ocultarse. Utilice para ello el menú o haga clic en la "x" de la esquina superior derecha de la ventana del recuadro.

  • Tecla rápida: Alt+V seguido de P
  • Menú: Vista -> Panel Lateral

Fijar y mover

Es posible mover el recuadro a cualquier punto de la pantalla haciendo clic con el ratón y, sin soltar el botón, arrastrar su barra de título. Arrástrelo hacia un extremo para fijarlo a la izquierda, a la derecha, en la parte superior o la inferior. Para ajustar el tamaño, pulse cerca de cualquier esquina y arrastre hasta conseguir el tamaño deseado.

Saltando elementos especiales

Se pueden evitar determinados elementos (estructuras anidadas como las tablas, o información complementaria como los recuadros). Esto significa que podrá seguir leyendo la entrada que precedía a la estructura ignorada.

  • Tecla rápida: Escape
  • Menú: Reproducir -> Escape

Elementos evitables

Una característica de los libros DAISY es que pueden contener elementos evitables u opcionales. Un elemento evitable es aquel que puede ser activado o desactivado en el libro. Ejemplos de posibles elementos evitables son los números de página y las notas a pie. Si no quiere escuchar estos elementos mientras lee, deberá asegurarse de que estén marcadas en el diálogo "Elementos Saltables".

  • Tecla rápida: Alt+I seguido de E
  • Menú: Información -> Elementos Opcionales...

Resumen informativo

AMIS proporciona tres tipos de resúmenes informativos: Sumario completo, Posición actual y Opciones de navegación. Todos ellos están disponibles como diálogos, accesibles a través del menú Info ("Alt + I")

Salir de AMIS

  • Tecla rápida: Alt+F4
  • Menú: Archivo -> Salir

Añadidos

AMIS se puede personalizar mediante el añadido de paquetes, tales como paquetes de idiomas o plug-ins.

Plug-ins

AMIS soporta plug-ins, que son interfaces alternativas de solo software o de software y hardware. Por ejemplo, joysticks, líneas braille o menús de escaneado. Todos los plug-ins se encuentran disponibles en nuestra web (http://amis.sf.net).

Paquetes de Idiomas

AMIS soporta UNICODE y es posible configurarlo para utilizar distintos paquetes de idiomas. Todos los paquetes de idiomas están disponibles en nuestra página web (http://amis.sf.net).
